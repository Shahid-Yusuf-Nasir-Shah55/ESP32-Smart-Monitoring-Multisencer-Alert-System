# ESP32 Smart Monitoring & Alert System

An IoT-based smart monitoring system built using ESP32 that detects environmental and security risks using multiple sensors and generates real-time alerts.

---

## 🚀 Features
- Real-time environmental monitoring
- Multi-sensor risk analysis
- OLED live data display
- Motion and intrusion detection
- Gas leakage monitoring
- Smart buzzer alert system

---

## 🧰 Hardware Used
- ESP32 DevKit V1
- DHT22 Temperature & Humidity Sensor
- PIR Motion Sensor
- MQ Gas Sensor
- Ultrasonic Sensor (HC-SR04)
- LDR Sensor
- HX711 Load Cell
- MPU6050 Accelerometer & Gyroscope
- SSD1306 OLED Display
- Buzzer

---

## 🧠 Working Principle
Sensor data is collected continuously and processed to calculate a risk score.  
Based on risk level (SAFE / LOW / MEDIUM / CRITICAL), the system activates alerts and displays status on OLED.
